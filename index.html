Java Core

Начинаете изучать язык Java? Тогда вам сюда - на ExamClouds вы найдете полный список лекций по изучению Java Core.

Каждый урок содержит домашние задания, вопросы и тесты по пройденным темам.

Урок 1 - Особенности Java. История Java. Java SE/EE/ME. JVM, JRE, JDK. Компилятор и интерпретатор. Байт-код. Создание первого приложения. Компиляция и запуск из консоли.

Урок 2 - Установка и начало работы с IntelliJ IDEA. Лексика языка. Примитивные типы. Рекомендации по именованию идентификаторов. Литералы и переменные.

Урок 3 - Преобразование примитивных типов. Консольный ввод/вывод.

Урок 4 - Операции.

Урок 5 - Управляющие операторы - ветвления - if, switch; цикла - while, do/while, for; перехода - break, continue, return. Массивы. Алгоритмы сортировки.

Урок 6 - VСS, git и GitHub.

Урок 7 - Введение в ООП.

Урок 8 - ООП.

 

Код с примерами на GitHub: https://github.com/milkina/java-core.git
